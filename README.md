📊 Madhav E-Commerce Sales Dashboard

This project presents an interactive E-Commerce Sales Dashboard built to analyze and visualize business performance across multiple dimensions such as sales, profit, customers, states, and product categories.

🚀 Project Overview

The dashboard provides a comprehensive view of sales data, helping stakeholders make data-driven decisions. It highlights key performance indicators (KPIs) and trends using visually appealing charts and graphs.

🎯 Key Features
📌 KPI Metrics
Total Sales Amount: 438K+
Total Quantity Sold
Average Order Value (AOV)
Overall Performance Summary
📈 Profit Analysis
Monthly profit trends (positive & negative variations)
Seasonal performance insights
🌍 State-wise Sales Analysis
Top-performing states like Maharashtra, Madhya Pradesh, etc.
Regional comparison of revenue
🛍️ Category & Sub-Category Insights
Sales contribution by categories (Saree, Bookcases, Printers, etc.)
Identification of high-profit products
💳 Payment Mode Distribution
COD, UPI, Credit Card, EMI, Debit Card usage breakdown
👥 Customer Insights
Top customers based on purchase value
🔍 Filters & Interactivity
Quarter-wise filtering (Q1–Q4)
State-level filtering for deeper analysis

📊 Insights Gained
Maharashtra is the highest revenue-generating state
Printers and Bookcases contribute significantly to profit
Profit fluctuates across months, with peaks in January & November
COD is the most used payment method
Certain months show negative profit, indicating cost or demand issues

Improve data storytelling skills
Support strategic business decisions
